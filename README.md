## Google Play Store Data Analysis (EDA)

### Overview

This project performs **Exploratory Data Analysis (EDA)** on a dataset of **Google Play Store applications** to uncover key trends and patterns. The analysis focuses on understanding how factors such as category, installs, ratings, and type (free/paid) influence app performance.

### Tools & Libraries

* **Python**
* **Pandas** – Data cleaning and manipulation
* **Seaborn & Matplotlib** – Data visualization
* **Jupyter Notebook** – Analysis environment

### Key Highlights

* Cleaned raw data by handling **missing values**, **duplicates**, and **inconsistent categories**.
* Analyzed relationships between **app ratings, reviews, installs, and categories**.
* Visualized insights such as **top-performing app categories** and **comparisons between free and paid apps**.

### Outcome

Generated actionable insights to help app developers and businesses optimize their strategies and improve user engagement.
